# Computer_Vision_Assignment_01

This repository contains my solutions for Computer Vision Assignment 01.

The assignment is available in two formats:

- Separate Python files for each question: `src/q1.py` to `src/q12.py`
- A single runnable Jupyter notebook: `assignment_01_notebook.ipynb`

## Running the Python Files

Install the required packages:

```powershell
py -3 -m pip install -r requirements.txt
```

Run the full assignment:

```powershell
py -3 src\main.py
```

Run an individual question:

```powershell
py -3 src\q7.py
```

Replace `q7.py` with any other question file as needed.

## Running the Jupyter Notebook

Open and run:

```powershell
jupyter notebook assignment_01_notebook.ipynb
```

The notebook contains all questions in one place and displays the relevant outputs after each question.

Outputs are written under `outputs/q1` through `outputs/q12`.

The generated HTML version of the notebook is `assignment_01_notebook.html`, which can be opened in a browser and printed/exported as a PDF.

## Notes

For Q7, the code uses the provided large/small image pairs inside `images/q7_images`.
